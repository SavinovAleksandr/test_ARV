================================================================================
  Модуль оценки АРВ — автономная сборка (без установки .NET Framework)
================================================================================

ЭТА ПАПКА СОДЕРЖИТ ВСЁ НЕОБХОДИМОЕ ДЛЯ ЗАПУСКА НА WINDOWS БЕЗ УСТАНОВКИ
.NET FRAMEWORK. Встроена среда выполнения .NET 8.

КАК ЗАПУСТИТЬ
--------------
1. Скопируйте ВСЮ папку Publish_Standalone на компьютер с Windows.
2. Запустите файл:  Модуль оценки АРВ.exe
   (двойной щелчок или из командной строки).

НИ .NET Framework, ни отдельная установка .NET на целевом ПК не требуются.

СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------
- Windows (x64)
- Для полной работы расчётов: RUSTab (ПК RUSTab) и Microsoft Word
- Шаблоны Rastr в папке пользователя: Documents\RastrWin3\SHABLON\

ПОВТОРНАЯ СБОРКА ЭТОЙ ПАПКИ (на машине с .NET SDK)
---------------------------------------------------
  dotnet publish avr_mu_computation_net8.csproj -c Release -r win-x64 --self-contained true -o Publish_Standalone

Исходный проект: avr_mu_computation_net8.csproj в корне репозитория.
